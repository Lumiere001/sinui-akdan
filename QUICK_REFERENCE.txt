GAME FLOW:
  1. Player joins with password → validates, sets representative flag
  2. Admin starts timer → broadcasts step 1 hint + locations
  3. Team navigates to location
  4. 3+ members within 50m → player checks location
  5. GPS proximity ≤40m → location validated
  6. Correct location → advance to next step + new hint
  7. Wrong location → show wrong photo, stay on step
  8. After step 3 correct → team complete + final photo
  9. Player submits pledge → stored in persistence
  10. Rep messages admin → broadcast to team + admin
